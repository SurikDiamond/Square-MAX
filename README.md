Privacy Policy

This Chrome extension is used as an internal tool to enhance workflow within specific web applications.

Data Collection:
The extension accesses and processes website content (such as page text and appointment data) locally in the user's browser to provide its functionality.

Data Usage:
All data is used only within the extension to modify the user interface and automate workflows. No data is sent to external servers.

Data Sharing:
The extension does not sell, transfer, or share user data with any third parties.

Storage:
The extension may store user preferences and settings locally using Chrome storage.

Security:
All processing happens locally in the browser environment.
